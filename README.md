# next-admin

nextjs 创建中后台项目

## 特性

1. 用户管理

   - 用户注册
   - 用户验证
   - 权限校验

## 环境依赖

- node v18 or above
- yarn

## 安装

```shell
# 1. 克隆这个仓库
git clone https://github.com/chengeheng/next-admin.git

# 2. 安装依赖
cd next-admin
yarn install
```

## 相关技术

- Typescript
- Nextjs
- Express
- React
- Antd

## 使用

1. 在开发模式下启动

```shell
npm run dev
```

2. 访问 https://localhost:8000
