# next-admin

Create middle-and-back-office projects using nextjs

## Features

## Dependencies

- node v18 or above
- yarn

## Installation

```shell
# 1. clone the repository
git clone https://github.com/chengeheng/next-admin.git

# 2. install dependencies
cd next-admin
yarn install
```

## Technologies Used

- Typescript
- Nextjs
- Express
- React
- Antd

## Usage

1. start the develop

```shell
npm run dev
```

2. open the application https://localhost:8000
